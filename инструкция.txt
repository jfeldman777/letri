1) создайте оглавление в котором вы будете работать
2) откройте интерфейс "командная строка" и перейдите в это оглавление = черный ящик
3) проверьте что у вас установлен питон
>python --version
Если нет - установите

4) проверьте, что у вас установлен GIT
> git --version
Если нет - установите

5) откройте интерфейс gitshell
перейдите в папку которую вы создали
с помощью команды CD = синий ящик

6)в черном ящике создайте в этой папке виртуальную среду
> python -m venv .
проверьте результат с помощью команды 
> dir
у вас появятся оглавления
Scripts
Lib
Include

7) перейдите в оглавление Scripts и запустите Actvate
> cd Scripts
> activate
вернитесь обратно в рабочую папку.
черный ящик выглядит теперь иначе
проверьте что вам доступен например PIP
> pip --version

8)в черном ящике инициализируйте GIT
> git init

9) в синем ящике скачайте код
> git pull https://github.com/jfeldman777/letri.git

убедитесь что все на месте - в черном ящике
> dir
появился директорий 
letri

10) в черном ящике восстановите среду обитания
> cd letri
> pip install -r requirements.txt
я не дождался конца установки звкрыл старый черный ящик и открыл новый

11) восстановите базу данных
> python manage.py migrate

12) запустите сервер
> python manage.py runserver

13) В браузере откройте адрес
 http://localhost:8000/
 Сайт откроется
 
 14) Остановите сервер команедой Ctrl-C
 
15) Создайте суперюзера
python manage.py createsuperuser

Задайте имя
почту
пароль (два раза)

16) запустите сервер.
Откройте браузер
водите в LOGIN
и введите реквизиты SUPERUSERа

17) Откроется форма
Но вы  - не трогая форму - зайдите в ADMIN
Вы в панели администратора

18) Чтобы новая версия оказалась на сервере
 вам надо научиться работать с удаленным сервером
 
 После того как новая версия окажется на GITHUB
 Из окна PUTTY надо сделать 
git pull 
и перегрузить сервер
